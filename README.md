🔹 Technologies & Features:
•React.js + React Router DOM for smooth navigation
•Context API for global state management
•Node.js + Express.js REST API for scalable backend services
•MongoDB for flexible and efficient data handling
•Scalable, clean architecture with modular code design
•Modern UI & Admin Dashboard (two live web apps)
•Full functionality: Authentication, cart, orders, payments, product management & more
•Thoroughly tested before deployment
•Deployed on Vercel for fast, reliable hosting

✨ This project reflects not just my technical growth but also my passion for building scalable, real-world applications.

 1️⃣ Admin Panel
Login as Admin

2️⃣ User Frontend (Moonlight)
User Signup & Login
Browse products with smooth UI

Continue payment with 3 flexible options:
 ✅ Stripe
 ✅ Razorpay
 ✅ Cash on Delivery (COD)
